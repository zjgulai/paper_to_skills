1. 解决的问题

场景一：全品类 VOC 周报生成

2. 核心算法逻辑

Subagent Decomposer 负责将复杂的 Task Blueprint 分解为可独立执行的子任务，并分配给专门的子 Agent。核心洞察：复杂任务（如"生成全品类 VOC 周报"）无法由单个 Agent 高效完成，必须分解为并行/串行的子任务，每个子任务由最优技能的子 Agent 执行。

  • 深度分析从 3-5 天缩短到 2-4 小时
