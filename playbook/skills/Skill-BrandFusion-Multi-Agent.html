1. 解决的问题

场景："Babycare" 品牌在 TikTok 的品牌一致性视频

2. 核心算法逻辑

首篇专注"T2V 品牌无缝植入"的论文。核心问题是：用 AI 生成品牌视频时，品牌 Logo/包装/视觉资产在视频中会变形、消失或被遮挡。BrandFusion 用 5 个 Agent 协同迭代优化，确保品牌元素自然融入视频。
