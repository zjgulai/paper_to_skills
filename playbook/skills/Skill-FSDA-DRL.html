FSDA-DRL 快慢双智能体动态定价与补货联合优化

Skill-FSDA-DRL · 04-供应链 · status=unknown

causalexperimentforecastingoptimizationmulti_agentfraud_detectionpricing广告与投放供应链与补货MAS与智能体工程定价与利润风控与合规WF-A 智能补货WF-B 广告优化

1. 解决的问题

场景1：Prime Day 备货 + 动态定价联合决策（Momcozy 母婴器械）

2. 核心算法逻辑

FSDADRL（FastSlow DualAgent Deep Reinforcement Learning）用两个独立的 RL 智能体，在不同时间频率上分别解决"定价"（快决策）和"补货"（慢决策）问题，并通过共享环境状态让它们协作而非博弈。

5. 业务场景

  • 定价 Agent：每天输出最优折扣率（可接入 Amazon Repricer API 自动执行）
  • 补货 Agent：每周输出补货建议量（与 ERP 采购模块对接）
  • 仿真报告：大促全周期的利润预测曲线与库存消耗预测
  • 历史同类商品季节性折扣曲线
  • 海运/空运 Lead Time 分布（均值 + 标准差）
  • FBA/海外仓当前库存 + 在途库存
  • 补货 Agent（慢尺度 = 月维度）：最优发船时间点 + 发货量
  • 定价 Agent（快尺度 = 日维度）：清仓价格路径（随到货日临近逐渐上调）
