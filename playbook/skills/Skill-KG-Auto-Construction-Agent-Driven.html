1. 解决的问题

场景一：从 Amazon 商品描述自动构建母婴商品知识图谱

2. 核心算法逻辑

传统知识图谱构建依赖人工定义 Schema 和编写抽取规则，成本高、扩展性差。AI Agent 驱动的 KG 自动构建 将全流程拆解为三个由 LLM Agent 协作完成的阶段，从非结构化产品描述中自动产出结构化知识图谱，无需预定义 Schema 或人工规则。
