WF-H 复购增长

从用户价值分层到精准干预再到 LTV 提升的端到端用户复购决策链，覆盖用户分层、流失预警、干预策略和 Review 激活四个阶段。

适用角色用户增长负责人运营负责人CMO
入口问题：你的复购率和用户留存当前最大挑战是什么？

预期收益

  • [OK] 可干预流失用户识别精准率 +25%，优惠券 ROI 提升 3-5x
  • [OK] 老客复购率提升 10-15%，LTV 增加
  • [OK] Review 获取率提升 30%（瞄准最优触达时机），评分质量更真实
  • [OK] 年化节省无效促销成本 25-50 万元（不再给"必然留存"用户发无谓优惠券）
用户价值分层
是否已建立系统的用户分层体系？
母婴用户生命周期明确（备孕→孕期→婴儿期→幼儿期），不同阶段需求差异巨大，统一运营会造成大量资源浪费
RFM Customer Segmentation for Targeted MarketingR/F/M 三维分层把用户分成 8 类（冠军→流失→沉睡），每类对应不同运营策略 LTV预测 - 零膨胀对数正态模型 (ZILN)零膨胀对数正态 LTV 预测，识别高潜力用户优先投入资源 STAN 用户生命周期自适应建模时空注意力模型判断用户当前所在生命周期阶段（备孕/婴儿期/幼儿期）
流失预警与识别
如何识别即将流失的用户？
母婴用户的自然流失率高（孩子大了就不需要了），关键是区分"自然毕业"和"可干预流失"
个体级精准流失预测（精准发券干预）
Uplift Modeling for Churn PredictionUplift 流失建模：识别"可说服者"，避免对必然流失/必然留存用户浪费资源 Customer Churn Prediction (用户流失预测)深度学习流失预测，捕捉复杂行为序列模式
群体级流失分析（制定策略方向）
Cohort Retention Analysis for User Lifecycle队列留存分析，找到复购黄金时间窗口（如首单后 14 天是关键节点） User Funnel and Behavior Path Analysis购买漏斗分析，定位最大流失节点
精准干预策略
选择干预方式？
干预不是越多越好——对错误用户干预会适得其反，"不要打扰者（Sleeping Dogs）"被优惠券刺激反而加速离开
发券/促销干预（节省预算最大化复购）
Guardrailed CATE-NBA预算护栏约束下的最优行动决策，自动输出最优干预名单（ROI 25-28万/月） Personalized Promotion Targeting — 个性化促销定向：用户响应异质性建模个性化促销定向，识别对哪种优惠响应最强的用户（ROI 300-600万）
内容/推荐干预（提升连带购买）
LLM Augmented Recommendation — 大语言模型增强个性化推荐：自然语言驱动的跨域用户意图理解LLM 驱动个性化推荐，理解用户自然语言表达的偏好（ROI 100万） Shopping Companion — 记忆增强的长期偏好购物 Agent跨会话长期偏好记忆，精准复购触达
Review 获取与品牌口碑激活
如何在正确时机触达用户获取真实评论？
评论数量和评分是 Amazon 排名的核心因素，主动在最佳时机请求评论可将 Review 获取率提升 30%+
Session意图漂移建模 - 跨会话用户购买意图变化检测用户意图转变检测，识别满意度高适合请求评论的时机 Emotional AI Customer Care — 情感感知客服：高压场景的同理心 AI情感感知的主动触达，在正确情绪状态下发送 Review 邀请 MAA 多 Agent 行动建议 - 从评论到产品改进决策链评论→产品改进决策链，让每条 Review 都变成产品迭代信号（ROI 510-920万/年）
复购增长效果验证
用户分层和精准干预启动后，如何量化复购率改善的真实贡献？
30天/90天追踪干预用户 vs 对照用户的复购率差异，形成 LTV 改善数据
Cohort Retention Analysis for User Lifecycle对比干预组 vs 对照组的90天留存率差异（严格因果验证） LTV预测 - 零膨胀对数正态模型 (ZILN)更新 LTV 预测模型，追踪干预后用户价值变化 Guardrailed CATE-NBA分析哪类干预对哪类用户最有效，优化下一轮干预策略
