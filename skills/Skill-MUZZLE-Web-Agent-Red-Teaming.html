1. 解决的问题

场景一：导购 Agent 商品描述注入（完整性攻击）

2. 核心算法逻辑

间接 Prompt Injection（IPI） 的核心机制：攻击者无法直接访问 Agent 的系统提示，但可通过控制 Agent 抓取的外部内容（商品描述、用户评论、网页正文）向 Agent 上下文注入恶意指令，使 Agent 偏离原始任务目标执行攻击者意图。
