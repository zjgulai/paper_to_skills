4. 输出

  • `signal_strength >= 0.8` → 高频模式（5分钟/次），大促当天、黑五
  • `signal_strength 0.5-0.8` → 中频模式（15-30分钟/次），大促前 2-3 天
  • `signal_strength < 0.3` → 低频模式（120分钟/次），平静期
  • `trend_score >= 0.75` → 立即启动专项爬取 + 通知选品团队
  • `trend_score 0.5-0.75` → 加入观察列表，持续监控 3-7 天
