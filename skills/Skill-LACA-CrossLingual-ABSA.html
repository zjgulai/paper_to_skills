2. 核心算法逻辑

跨语言 ABSA 痛点:目标语言(德/法/西/日)无标注数据,翻译方法会丢失 aspect 词对齐. LACA 用逆向去噪思路:① 用英文标注模型对目标语言文本做零样本预测 → ② 把伪标签反向喂给 LLM,让 LLM 生成与该标签匹配的干净目标语言句子 → ③ 合并英文真标注 + LLM 生成伪标注重训模型. 绕过 MT 对齐错误,跨语言 ABSA SOTA.
