2. 核心算法逻辑

ReAct (Reasoning + Acting) 提出了一种推理与行动交织的范式。核心洞察：纯推理（ChainofThought）容易幻觉，纯行动（Tool Use）缺乏规划——只有把两者交替进行，才能既保持思维连贯性又确保信息准确性。
