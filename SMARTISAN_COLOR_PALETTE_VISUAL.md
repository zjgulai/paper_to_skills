## 语义色彩映射表

### 背景层级
```
主背景       #ffffff  (--bg-primary)
次背景/卡片  #f7f7f7  (--bg-secondary)  ← 微妙对比，高端感来源
三级/Hover   #f0f0f0  (--bg-tertiary)
遮罩         rgba(26,26,26,0.5)  (--bg-overlay)
```

### 文本层级
```
主文本       #1a1a1a  (--text-primary)    标题/重要内容 ← 4.5:1 对比度
次文本       #2d2d2d  (--text-secondary)  描述/次要内容
三级文本     #4a4a4a  (--text-tertiary)   辅助/标签
禁用文本     #666666  (--text-disabled)   禁用状态
反色文本     #ffffff  (--text-inverse)    深背景上的文本
```

### 边界和分割
```
主边框       #999999  (--border-primary)
次边框       #cccccc  (--border-secondary)
浅边框       #e0e0e0  (--border-light)     ← 最常用，卡片/输入框
```

### 强调和交互
```
强调主色     #0073e6  (--accent-primary)   按钮/链接 ★
悬停强调     #1a85ff  (--accent-hover)     +18% 亮度
禁用强调     #4da6ff  (--accent-disabled)  40% 不透明
```

## 色彩对比度验证（WCAG AA）

### 通过的组合 ✅
```
#1a1a1a 文本 on #ffffff 背景  → 对比度 11.7:1  超级易读
#2d2d2d 文本 on #ffffff 背景  → 对比度 9.5:1   易读
#1a1a1a 文本 on #f7f7f7 背景  → 对比度 9.2:1   易读
#666666 文本 on #ffffff 背景  → 对比度 5.2:1   AA 通过 ✓
#0073e6 按钮 on #ffffff 背景  → 对比度 4.8:1   AA 通过 ✓
#ffffff 文本 on #0073e6 背景  → 对比度 6.2:1   AA 通过 ✓
```

### 需要避免的组合 ❌
```
#999999 文本 on #ffffff 背景  → 对比度 3.5:1   AA 失败（标题可用）
#e0e0e0 文本 on #ffffff 背景  → 对比度 1.1:1   完全不可读（仅用作边框）
```

## 深色模式参考（可选扩展）

如果未来需要支持深色模式，建议这样映射：

```css
@media (prefers-color-scheme: dark) {
  :root {
    /* 深色模式下重新定义语义色彩 */
    --bg-primary:       #1a1a1a;     /* 深背景 */
    --bg-secondary:     #2d2d2d;     /* 卡片 */
    --text-primary:     #ffffff;     /* 浅文本 */
    --text-secondary:   #e0e0e0;
    --border-light:     #4a4a4a;     /* 深边框 */
    --accent-primary:   #1a85ff;     /* 稍浅的蓝 */
    
    /* 其他颜色维持不变 */
  }
}
```

## 色彩系统的设计哲学

### 为什么是灰色而不是其他中性色？
- **灰色**：无饱和度，显得高级、克制
- 棕色：显得复古、温暖
- 蓝灰：显得冷硬、科技
- 纯黑白：对比太强，容易疲劳

### 为什么背景是 #f7f7f7 而不是 #f5f5f5？
- `#f7f7f7` = rgb(247,247,247) → 几乎白但有温度
- `#f5f5f5` = rgb(245,245,245) → 过于接近纯白，分层感不足
- `#fafafa` = rgb(250,250,250) → 介于两者，也可用

### 为什么文本是 #1a1a1a 而不是 #000000？
- 纯黑 `#000000` 对眼睛冲击强，现代设计避免
- 深灰 `#1a1a1a` (rgba(26,26,26)) 更温和，易读
- 仅在极端强调时才用纯黑

### 为什么品牌色是 #0073e6 而不是 #0066cc？
- `#0073e6` (RGB 0, 115, 230) = 更高亮度，更科技
- `#0066cc` = 略微冷沉，传统感
- `#007bff` = Bootstrap 默认，太平凡
