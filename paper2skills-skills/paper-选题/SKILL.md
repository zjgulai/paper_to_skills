# paper-选题

从 ArXiv 筛选高质量学术论文的技能。

## 概述

根据母婴出海跨境电商业务需求，从 ArXiv 筛选相关的学术论文。

## 触发方式

用户提及以下内容时触发：
- "筛选论文"
- "选择论文"
- "搜索论文"
- "找 paper"
- "论文选题"
- "arXiv 搜索"

## 筛选标准

### 领域选择

根据业务需求选择以下领域之一：

1. **因果推断** - 广告归因、促销效果评估、用户分层运营
2. **A/B实验** - 转化率测试、定价策略验证、UI改版效果
3. **时间序列** - 销量预测、需求预测、库存计划
4. **供应链** - 选品决策、备货策略、物流调度
5. **推荐系统** - 复购推荐、搜索排序、首页推荐
6. **增长模型** - 拉新、促活、留存、变现

### 论文评估标准

- **实用性**：论文提出的方法是否能在6个月内落地
- **可操作性**：是否有现成的工程实现
- **数据依赖**：需要什么样的数据，数据是否容易获取
- **业务价值**：能解决什么业务问题，预期收益多大

### 论文质量筛选（必须）

#### 必须排除的类型

以下论文类型应直接排除：

- **Survey/Review/Overview**: 文献综述类论文，缺乏原创性贡献
- **Theoretical (无实验)**: 纯理论推导但没有实验验证的论文
- **Meta-analysis**: 元分析类论文
- **Position Paper**: 立场声明类论文

#### 必须满足的条件

优先选择包含以下内容的论文：

- **Algorithm/Pseudo-code**: 有明确的算法描述或伪代码
- **Experiments**: 有完整的实验设计和结果
- **Implementation**: 有开源代码或工程实现

#### 质量评分标准 (1-10分)

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 算法创新性 | 20% | 1-10分，核心算法是否有创新 |
| 实验完整性 | 30% | 1-10分，是否有充分的实验验证 |
| 工程可落地性 | 30% | 1-10分，代码是否可运行，数据依赖是否可满足 |
| 业务适配度 | 20% | 1-10分，与母婴出海业务的契合程度 |

**评分 >= 7分** 的论文才推荐萃取。低于7分的论文建议：
- 暂不萃取
- 或标记为"待观察"，等待更合适的版本

### AI方法与6大专题结合

将AI/ML/DL方法作为解决6大专题问题的重要方案来源：

| AI方法 | 可解决的6大专题问题 |
|--------|---------------------|
| **LLM/大语言模型** | 因果推断(反事实推理)、推荐系统(对话推荐)、增长模型(智能客服) |
| **深度学习(DNN/CNN)** | 时间序列预测、需求预测、用户行为建模 |
| **强化学习(RL)** | 动态定价、A/B实验(bandits)、库存优化、推荐策略优化 |
| **图神经网络(GNN)** | 用户关系建模、供应链网络优化、推荐系统 |
| **迁移学习/多任务学习** | 冷启动问题(推荐系统)、小样本学习(增长模型) |

**搜索策略**：
- 使用 `[AI方法] + [专题场景]` 组合关键词
- 例如：`transformer demand forecasting`、`reinforcement learning pricing`、`LLM causal inference`

## 搜索方法

### 1. ArXiv 搜索

使用 ArXiv API 或网页搜索：

```bash
# 搜索示例
curl "https://export.arxiv.org/api/query?search_query=all:uplift+modeling&start=0&max_results=10"
```

### 2. 论文筛选

根据以下标准筛选：

- 发表时间：优先选择近3年的论文
- 引用量：有一定引用量说明经过验证
- 代码实现：有开源代码的优先
- 业务适配：能与母婴出海业务场景结合

## 输出格式

### 论文信息

输出筛选后的论文信息：

```
## 论文标题

- **arXiv ID**: xxxx.xxxxx
- **作者**: xxx, xxx
- **日期**: YYYY-MM-DD
- **领域**: [领域名称]
- **摘要**: [论文摘要]
- **业务适配度**: ⭐⭐⭐⭐⭐
- **推荐理由**: [为什么推荐]
```

## 注意事项

- 优先选择有工程实现的论文
- 优先选择与母婴出海业务相关的论文
- 避免选择过于理论化的论文
- 评估数据依赖，确保数据可获取
