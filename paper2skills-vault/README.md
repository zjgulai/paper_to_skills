# paper2skills 知识库

## 项目简介
将前沿学术论文转化为可落地的商业决策技能卡片，专注于母婴出海跨境电商业务场景。

## 目录结构

### 技能领域
- `01-因果推断/` - 因果效应估计、无偏估计方法
- `02-A_B实验/` - 实验设计、统计功效、置信区间
- `03-时间序列/` - 预测精度、不确定性量化
- `04-供应链/` - 库存优化、成本控制、需求匹配
- `05-推荐系统/` - 召回率、排序质量、个性化程度
- `06-增长模型/` - 漏斗转化、LTV、用户生命周期
- `07-NLP-VOC/` - 情感分析、客户声音、舆情挖掘
- `08-知识图谱/` - 异构表示学习、层次嵌入、语义检索
- `09-DataAgent-LLM/` - DataAgent、LLM 数据分析
- `10-MAS/` - 多智能体协作、任务规划、编排执行
- `11-AI人文/` - AI 技术与人文跨域类比
- `12-ML基础/` - 特征工程、模型评估
- `13-广告分析/` - 归因建模、ROAS 优化
- `14-用户分析/` - 漏斗、留存、RFM
- `15-营销投放分析/` - MMM、促销因果效应
- `16-智能体工程/` - Agent Skills / Context / MCP / 协议落地

### 资源库
- `07-资源库/MasterPrompt.md` - Master Prompt 完整内容
- `07-资源库/关键词库.md` - 论文搜索关键词

## 使用方法
1. 从 ArXiv 筛选论文
2. 使用 Master Prompt 生成 Skill 卡片
3. 质量审核后同步到各平台
