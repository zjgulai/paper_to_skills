## 三、已知限制与后续建议

1. **U002 无 Persona**：U002 仅购买了 S9_便携款 + 车载充电器，但 mock LLM 未触发规则，导致随机游走无标签可传播。生产环境接入真实 LLM API 后可解决。
2. **小规模图传播衰减**：5 用户演示数据图稀疏，部分用户（U002）Jaccard 相似度为 0，随机游走无法传播。实际 10 万+用户图传播效果大幅改善。
3. **建议**：接入生产环境时替换 `_mock_llm_label` 为真实 LLM API（推荐 GPT-4o-mini 或 Qwen2.5-72B）；Persona 集合由业务团队初始化后固化复用。
