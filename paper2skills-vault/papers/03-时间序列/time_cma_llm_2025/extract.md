## 核心算法提炼

### 算法名称
TimeCMA (Time Series Cross-Modality Alignment) / 时序-文本跨模态对齐大模型预测框架

### 核心思想
传统的时间序列预测模型（无论统计学还是深度学习）参数量小、训练数据有限，难以学习到复杂的周期性和外部事件。近年来，将**时间序列转为文本 Prompt 输入给大语言模型（LLM）**成为了一个巨大风口，因为 LLM 见识过海量的互联网人类知识规律。
但现有的直接拼接方案存在缺陷：LLM 生成的 Embedding 往往是“纠缠（Entangled）”的，即数值序列本身的纯净周期信号，被文本提示词（Prompt）中强烈的语义信号掩盖和污染了。

**TimeCMA 提出了“双路编码 + 跨模态对齐”的优雅架构**：
1. **双分支编码**：
   - *时序特征分支*：用轻量级网络单独处理纯数字序列，提取出“纯净但较弱”的周期特征。
   - *LLM 提示分支*：把时序包裹在 Prompt 里丢给冻结的 LLM，提取出“鲁棒但纠缠”的深层语义特征。
2. **跨模态对齐检索（Cross-Modality Alignment）**：利用对比学习思路，在特征空间里将这两种 Embedding 相互对照。让 LLM 的特征“教会”纯数字特征拥有大局观，同时去除文本带来的无关噪声。提取两者的精华。
3. **“尾部Token压缩”黑科技**：由于 LLM 处理长序列极慢，TimeCMA 设计了一种特殊的 Prompt，强迫 LLM 把所有关于未来时序预测的关键信息浓缩到**输出的最后一个 Token** 里面。下游的预测头只读这一个 Token！极大地降低了计算成本，加快了推理速度。

### 为什么好用（优势）
1. **零/少样本（Zero/Few-shot）降维打击**：因为它底层是 LLM，即便在一条它从未见过的全新指标序列上，也能给出极高水准的预测。
2. **完美融合数值与语义（The best of two worlds）**：真正意义上的多模态预测，兼具传统算法捕捉局部波动的敏锐度，以及 LLM 对全局趋势（节假日、宏观规律）的理解力。
3. **工程上的极致优化**：通过尾Token压缩（Last token encapsulation），解决了 LLM 做时序预测推理太慢、显存太大的致命痛点，使其在生产环境中具备了极强的可落地性。

## 业务适配设计：重大事件大促期间的超预期销量预警

### 场景: 跨境电商在特殊节点（如超级碗、政策变动）的销量预测
在海外业务中，销量有时会被纯“文本事件”左右。比如在“超级碗”期间，某款零食销量可能暴增；或者当美国某个州出台了新的环保法案，某种塑料包装产品的销量立刻暴跌。
- **痛点**：传统数值模型（如 Prophet, TFT）只认识数字，它们会觉得这种突然的暴涨是“异常值”，不仅预测不出来，还会扰乱后续的基线拟合。
- **TimeCMA 方案落地**：
  - **输入构建**：不仅传入过去 30 天的销量数字，还在 Prompt 中加上一句话：“该商品主要销往加州，本周加州正式落地了限制塑料制品的法案1023，请基于此预测未来 7 天的销量走势。”
  - **跨模态推理**：LLM 凭借其通识知识网络，瞬间理解了“限制法案”对“塑料制品销量”的打压逻辑。TimeCMA 的跨模态对齐模块将这种高维的语义理解，精准映射到数值空间的下降斜率中。
- **预期价值**：赋予供应链大脑真正的“世界常识（World Knowledge）”。在面临新产品上线、突发公关事件、宏观政策切换等“无历史数据可依”的断崖式场景时，提供高准度的销量预测防御。
