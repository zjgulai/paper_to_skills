# Paper Extract: Beyond the Star Rating

## 论文信息
- **arXiv ID**: 2602.21082
- **标题**: Beyond the Star Rating: A Scalable Framework for Aspect-Based Sentiment Analysis Using LLMs and Text Classification
- **日期**: 2026-02-24
- **摘要**: 提出混合框架 - 使用 ChatGPT 识别评论中的关键方面(aspect)，然后用传统ML分类器进行大规模情感分类。在 470 万条餐厅评论上验证，显著提升了评分预测的准确性。

## 核心贡献
1. **混合架构**: LLM（ChatGPT）用于方面识别，传统ML用于情感分类
2. **大规模验证**: 在 470 万条真实评论上验证有效性
3. **回归分析**: 验证方面对评分的解释力（R² 显著提升）
