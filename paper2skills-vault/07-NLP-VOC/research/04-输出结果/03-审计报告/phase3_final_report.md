## 三、关键数据质量发现

### 3.1 Trustpilot 数据不是 Momcozy 官方评论

Trustpilot 74,007条零标签中：
- 仅 242 条（0.3%）提到 "momcozy"
- 其余是 Baby Birds、Formuland、Seraphine、Philips、Béaba、Darty 等品牌
- 法语评论中大量是法国本土母婴电商的评价

**建议**：Trustpilot 数据需要清洗，只保留 Momcozy 官方账号下的评论。如果当前数据已包含多品牌，建议：
- 方案A：用品牌名过滤，只保留 Momcozy 相关评论
- 方案B：将非 Momcozy 评论标记为"竞品/行业参考"，不计入覆盖率

### 3.2 Amazon竞品需要品线推断增强

Amazon竞品 84,454条零标签中：
- 0.4% 提到 "momcozy"（363条）
- 4.3% 含 "pump"（3,592条）→ 吸奶器
- 8.0% 含 "bra"（6,723条）→ 内衣服饰
- 13.4% 含 "baby"（11,296条）→ 通用母婴

**可通过产品关键词推断品线的约 20,000 条**，提升覆盖率约 5.5%。

### 3.3 Momcozy自有数据品线映射缺失

Momcozy 自有 16,931条零标签：
- 全部有 SPU 编码
- 但产品主数据中 85.3% 的 SPU 无 VOC 品线映射
- **这是数据质量问题，修复后可立即覆盖**

## 四、从 49.3% → 70% 的路径

| 改进措施 | 预期覆盖条数 | 覆盖率提升 | 优先级 | 难度 |
|----------|------------|-----------|--------|------|
| Amazon竞品关键词推断品线 | ~20,000 | +5.5% | P1 | 低 |
| Momcozy品线映射补全 | ~14,000 | +3.8% | P1 | 低 |
| Trustpilot Momcozy过滤 | ~200 | +0.1% | P2 | 低 |
| Zendesk邮件线程预处理 | ~3,000 | +0.8% | P2 | 中 |
| Reddit通用标签 | ~500 | +0.1% | P3 | 低 |
| **合计** | **~37,700** | **~+10.3%** | | |

**49.3% + 10.3% = 59.6%**，仍不达 70%。

要达到 70%，还需要：
- **ALCHEmist 74个标签部署**：覆盖有品线但零标签的约 30%（~27,000条）→ +7.4%
- **LLM辅助语义打标**：对长尾文本进行语义理解

**59.6% + 7.4% = 67.0%**，接近 70%。
