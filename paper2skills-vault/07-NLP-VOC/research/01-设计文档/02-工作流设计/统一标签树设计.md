## 一、多维标签空间架构

采用 TaxoAdapt `MultidimensionalTaxonomy` + AutoTag `LabelSystem` 融合架构：

```
评论输入 (英文文本 + 评分 + 数据源类型 + SPU编码)
    │
    ├──→ L0 路由: 数据源类型 → 决定处理策略
    │
    ├──→ 维度1: 品线维度 → [内衣服饰 / 吸奶器 / 喂养电器 / 智能母婴电器 / 孕产康 / 母婴综合护理]
    ├──→ 维度2: 品类维度 → [文胸 / 吸奶器 / 消毒器 / ... / 调奶器]
    ├──→ 维度3: AIPL旅程维度 → [A / I / P1 / P2 / L1 / L2 / L3]
    ├──→ 维度4: 问题类型维度 → [TPI / EPI / PPI / Orders&Logistics / ...]
    ├──→ 维度5: 业务分类维度 → [产品问题 / 订单及物流 / 信息咨询 / ...]
    ├──→ 维度6: 画像维度 → [55个原子标签]
    ├──→ 维度7: 情感维度 → [极性 + 强度]
    └──→ 维度8: 品牌维度 → [Momcozy提及 / 竞品提及 / 品牌对比]
```

**关键设计**: 8个维度独立打标，一条评论可同时打上多个维度的标签。

## 三、标签融合输出格式

每条评论打标后输出：

```python
@dataclass
class VOCLabels:
    """单条评论的完整标签集合"""
    review_id: str
    source_type: str           # L0: 退货留言/客服工单/商品评论/Trustpilot

    # 维度1-2: 产品维度
    product_line: str          # VOC品线
    category: str              # VOC品类
    spu_code: str              # SPU编码

    # 维度3: AIPL旅程
    aipl_stage: str            # 主阶段 A/I/P1/P2/L1/L2/L3
    aipl_signals: list[str]    # L2信号列表

    # 维度4-5: 问题/业务分类（现有标签直接映射）
    eng_classification: str    # classification-Tag
    cn_level1: str             # 标签一级分类
    cn_level2: str             # 标签二级分类
    cn_level3: str             # 标签三级分类
    voc_tag: str               # VOC标签（中文）

    # 维度6: 画像标签
    persona_tags: list[str]    # 55个原子标签中的命中项

    # 维度7: 情感
    sentiment_polarity: float  # -1.0 ~ +1.0
    sentiment_intensity: int   # -5 ~ +5
    aspect_sentiments: dict    # {方面: 情感分}

    # 维度8: 品牌
    brand_mentions: list[str]  # [Momcozy, Spectra, ...]
    brand_comparison: bool     # 是否涉及品牌对比

    # 质量评分
    quality_score: float       # ReviewQuality 综合分
    is_suspicious: bool        # 是否虚假评论
```

## 四、打标策略分路

| 数据源 | 数量 | 核心策略 | 主要产出维度 |
|--------|------|---------|------------|
| **退货留言** | 212,746 | 规则标注（结构化字段多） | 品线/品类/业务分类/情感 |
| **客服工单** | 124,928 | ALCHEmist + 规则（长文本） | 全维度（特别是AIPL-I/P1/L2） |
| **商品评论** | 15,418 | ABSA + 画像标签（有评分） | AIPL-L1/情感/画像/品牌 |
| **Trustpilot** | 2,605 | 轻量分类 + 情感（短评） | AIPL-L3/情感/品牌 |

## 五、与现有技能的衔接

| 维度 | 技能组件 | 复用方式 |
|------|---------|---------|
| 品线/品类 | 产品主数据表 | 直接字段映射 |
| AIPL旅程 | AIPL-VOC-Lifecycle-Tags | 关键词改为英文 |
| 问题类型 | 现有465标签 | 直接classification映射 |
| 业务分类 | 现有3级分类 | 直接字段映射 |
| 画像 | 55原子标签规则 | 直接复用英文关键词 |
| 情感 | ABSA-BERT-MoE + SentimentIntensity | 切换为英文模型 |
| 品牌 | 品牌实体识别 | 规则匹配竞品名单 |
| 质量 | ReviewQuality-Scoring | 适配英文特征 |
