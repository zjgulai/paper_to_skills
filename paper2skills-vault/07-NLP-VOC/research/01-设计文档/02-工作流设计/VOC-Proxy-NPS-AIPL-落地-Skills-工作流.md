## 三、四路自动打标策略

### 路1+2: 退货留言 + 客服工单 → 规则标注 + ALCHEmist

**合并原因**: 两者均来自 Zendesk，结构相似，可共用处理策略。

```python
def track_ticket_labeling(ticket_data):
    """
    1. 现有标签直接映射:
       - classification-Tag → 维度4(问题类型)
       - 标签一级/二级/三级分类 → 维度5(业务分类)
       - VOC标签 → 维度5叶子节点
       - SPU编码 → 维度1+2(品线/品类)

    2. 规则标注新维度:
       - AIPL旅程: 基于关键词规则 + 业务分类推导
         * 信息咨询 → A/I
         * 产品问题 → P1/P2/L1/L2
         * 售后咨询 → L2
       - 画像标签: 55个原子标签关键词匹配
       - 情感极性: 基于星级评分 + 情感词典
       - 品牌提及: 规则匹配

    3. ALCHEmist增强:
       - 用现有465标签作为种子，生成Label Function
       - 覆盖"未知"标签的数据（38.8%）
       - Data Programming投票聚合
    """
```

**预期覆盖率**:
- 维度4+5(现有标签): 100%（直接映射）
- AIPL旅程: 60%+（规则推导）
- 画像标签: 40%+（关键词匹配）
- 情感: 80%+（基于评分+词典）

### 路3: 商品评论 → ABSA + 画像标签 + 品牌

```python
def track_review_labeling(review_data):
    """
    1. ABSA方面情感提取（英文模型）:
       - 方面: 质量/物流/价格/服务/设计
       - 情感: -1~+1
       - 强度: -5~+5

    2. 画像标签提取:
       - 55个原子标签关键词匹配
       - 评论文本更长，画像标签覆盖率更高

    3. AIPL旅程推断:
       - L1-首用满意: 星级+正面词
       - L1-开箱反馈: 关键词 unboxing/packaging

    4. 品牌维度:
       - Momcozy提及检测
       - 竞品提及检测（Spectra/Medela/Willow/Elvie）
    """
```

### 路4: Trustpilot → 轻量分类 + 情感 + NPS

```python
def track_trustpilot_labeling(trustpilot_data):
    """
    1. AIPL-L3-推荐意愿检测（核心）:
       - 关键词: recommend, suggest, tell friends
       - 5星+推荐词 → 高推荐意愿
       - 1-2星 → 贬损者

    2. Proxy NPS计算:
       - 推荐者(9-10分/5星+推荐) - 贬损者(0-6分/1-2星)

    3. 情感极性:
       - 直接基于rating映射

    4. 品牌维度:
       - 竞品提及检测
    """
```

## 四、Phase 4: 融合对齐与指标计算

### 4.1 标签-指标映射矩阵

| 指标名称 | 依赖标签 | 数据源 | 计算方式 |
|---------|---------|--------|---------|
| **Proxy NPS** | L3-推荐意愿 + 情感极性 | Trustpilot+评论 | %推荐者 - %贬损者 |
| AIPL-A-认知转化率 | A-品牌搜索 → I-产品对比 | 工单 | 计数/占比 |
| AIPL-I-兴趣深度 | I-功能咨询 + I-价格关注 | 工单+评论 | 提及率/情感均值 |
| AIPL-P1-购买满意度 | P1-比价验证 + 方面情感 | 评论 | ABSA情感均值 |
| AIPL-P2-物流体验 | P2-物流体验 + Orders&Logistics | 工单+评论 | NPS公式 |
| AIPL-L1-首用满意度 | L1-开箱 + L1-首用 + 星级 | 评论 | 加权均值 |
| AIPL-L2-售后满意度 | L2-售后体验 + 售后咨询 | 工单 | 解决率/响应时长 |
| AIPL-L3-推荐率 | L3-推荐意愿 + L3-UGC | Trustpilot | 计数/占比 |
| 产品质量满意度 | 方面-质量 + EPI/PPI/TPI + ABSA | 全源 | ABSA情感均值 |
| 物流体验得分 | 方面-物流 + Orders&Logistics + ABSA | 工单+评论 | ABSA情感均值 |
| 画像渗透率 | 55原子标签 | 全源 | 标签覆盖率 |
| 画像-节点分布 | 画像标签 × AIPL节点 | 全源 | 交叉计数 |
| 三维交叉指数 | 画像 × 节点 × VOC标签 | 全源 | 共现计数/占比 |
