# VOC用户画像萃取体系 - 完整图谱

## 概述

本文档整合了VOC（Voice of Customer）用户画像萃取的完整技术体系，包含7个技能的联动关系，特别针对**Momcozy吸奶器**业务场景设计。

### 画像标签定义表

| 标签类别 | 标签值 | 数据来源 | 技能支撑 |
|---------|--------|---------|---------|
| **人群类型** | 职场背奶妈妈/全职新手妈妈/二胎经验妈妈/出差妈妈 | 评论关键词+搜索意图 | PERSONABOT |
| **使用场景** | 公司背奶/夜间家用/外出便携/医院开奶 | 评论内容分析 | TopicImpact |
| **核心痛点** | 噪音/续航/清洗/配件/吸力/舒适度 | 观点单元情感分析 | TopicImpact + Spiral of Silence |
| **决策因素** | 价格/品牌/功能/口碑/颜值 | 购买前搜索意图 | REVISION |
| **偏好特征** | 效率优先/静音优先/便携优先/性价比优先 | 评论情感倾向 | CSK聚类 |
| **生命周期** | 孕晚期/月子期/哺乳期早期/哺乳期中期/断奶期 | 购买时间+宝宝月龄 | SoMeR时间视角 |
| **社交属性** | 分享型/求助型/潜水型/KOC型 | 社交互动数据 | SoMeR网络视角 |

## 技能联动示例（Momcozy场景）

### 示例1: 个体用户画像生成流程

```
用户U12345的原始数据:
├── 搜索日志: "静音吸奶器推荐" → "Momcozy S12评测"
├── 评论1: "吸力很强但噪音大，在公司用尴尬"
├── 评论2: "配件清洗方便，小零件容易丢"
└── 客服对话: "请问有静音配件吗"

↓ REVISION分析
搜索意图 = 静音关注型 + 效率关注型

↓ Spiral of Silence分析
发现该用户的"配件困扰"意见属于少数派但被沉默

↓ TopicImpact提取观点单元
[吸力:9] [噪音:3] [便携:8] [配件:4]

↓ PERSONABOT生成画像
{
  "人群类型": "职场背奶妈妈",
  "核心痛点": ["噪音困扰", "配件管理"],
  "强需求": ["静音模式", "便携设计"],
  "画像总结": "效率导向的职场背奶妈妈，对噪音敏感"
}

↓ SoMeR多视角编码
嵌入向量 = [0.23, -0.15, 0.87, ...]
所属群体 = C1-效率背奶型 (35%用户)
相似用户 = 1245人

↓ 业务应用
推荐: 静音配件包 + 便携收纳袋 + 下一代静音款预售
```

### 示例2: 群体画像与营销策略

```
目标群体: 职场背奶妈妈
筛选条件: 评论含"上班/公司/背奶"关键词
群体规模: 35%用户基础 (约3.5万人)

↓ Spiral of Silence挖掘
发现沉默的痛点: 公司哺乳室条件差(62%)、吸奶时间与会议冲突(45%)

↓ TopicImpact观点分析
高频主题: 静音(78%)、便携(65%)、效率(58%)
情感分布: 静音主题负面占比高(avg=3.2)

↓ PERSONABOT群体画像
{
  "segment_name": "职场背奶妈妈",
  "core_needs": [
    {"need": "静音体验", "importance": 8.8},
    {"need": "便携性", "importance": 8.5},
    {"need": "效率", "importance": 8.2}
  ],
  "pain_points": [
    {"pain": "公司哺乳室条件差", "frequency": "62%"},
    {"pain": "吸奶时间与会议冲突", "frequency": "45%"}
  ],
  "marketing_insights": {
    "key_message": "10分钟高效背奶，静音不尴尬",
    "channel": "小红书/微博职场妈妈社群",
    "bundle": "主机+便携包+备用配件"
  }
}

↓ SoMeR群体聚类
在职场背奶妈妈群内细分为:
- C1-1: 静音敏感型 (12%)
- C1-2: 效率优先型 (15%)
- C1-3: 便携关注型 (8%)

↓ 精准营销
C1-1静音敏感型 → 推送降噪配件+静音款预售
C1-2效率优先型 → 强调10分钟快速吸奶
C1-3便携关注型 → 推荐轻便款+便携套装

预期效果: 营销转化率 +60%，客单价 +32%
```
