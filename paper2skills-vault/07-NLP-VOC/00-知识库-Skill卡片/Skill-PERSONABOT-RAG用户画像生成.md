## ① 算法原理

### 核心思想
传统用户画像是静态标签（如"25-35岁女性"），缺乏动态性和可解释性。PERSONABOT提出**RAG驱动的活文档画像**：通过检索真实用户评论作为上下文，LLM动态生成结构化、可溯源、可更新的用户画像。

### 数学直觉

**RAG检索**：
```
Retrieved Reviews = TopK(similarity(Query, Review Corpus))
```
基于用户ID或画像查询，检索最相关的历史评论。

**画像生成（Few-Shot + CoT）**：
```
Prompt = System Instruction + Few-Shot Examples + Retrieved Reviews + Output Schema
Persona = LLM(Prompt, temperature=0.3)
```

**画像Schema结构化**：
```json
{
  "demographics": {"age": "", "role": "", "experience": ""},
  "needs": ["need1", "need2"],
  "pain_points": ["pain1", "pain2"],
  "preferences": ["pref1", "pref2"],
  "usage_scenarios": ["scenario1", "scenario2"]
}
```

**反直觉洞察**：想象你问"谁是Momcozy的典型用户"，传统回答可能是"新手妈妈"。但PERSONABOT会检索真实评论，生成**活灵活现的画像**："28岁职场妈妈，宝宝6个月，每天背奶，对噪音敏感但注重效率，希望吸奶器能快速完成且不打扰同事"——这才是可指导产品设计的 actionable insight。

### 关键假设
1. 真实评论比统计标签更能还原用户全貌
2. LLM能从文本中提炼结构化画像属性
3. RAG检索能确保画像基于真实数据，非幻觉

## ② Momcozy吸奶器应用案例

### 场景1: 个体用户画像生成

**业务问题**  
用户U12345在Momcozy购买了S12吸奶器，留下了3条评论。如何基于这些评论生成她的完整画像，用于个性化推荐？

**数据输入**
```
用户U12345的评论：
1. "吸力很强，10分钟能吸150ml，适合我这种奶多的妈妈"
2. "就是噪音有点大，在公司用有点尴尬"
3. "配件清洗还算方便，就是小零件容易丢"
```

**RAG检索增强**
```
检索相似用户评论（同为职场背奶妈妈）：
- "便携性很好，放包里不占地方"
- "建议配个收纳袋，零件不容易丢"
- "静音模式其实很安静，可以试试"
```

**生成画像输出**
```json
{
  "user_id": "U12345",
  "demographics": {
    "role": "职场背奶妈妈",
    "experience": "奶量充足型",
    "usage_stage": "哺乳期中期"
  },
  "needs": [
    "高效吸奶（时间短）",
    "便携性（公司使用）",
    "配件管理方案"
  ],
  "pain_points": [
    "噪音困扰（公司场景）",
    "小零件易丢失"
  ],
  "preferences": [
    "吸力强劲",
    "清洗便利",
    "希望有静音模式"
  ],
  "usage_scenarios": [
    "工作日公司背奶",
    "每天2-3次"
  ],
  "persona_summary": "效率导向的职场背奶妈妈，对吸奶速度要求高，但受限于公司环境对噪音敏感，需要配件管理解决方案"
}
```

**业务应用**
- **个性化推荐**：推送静音配件包+便携收纳袋
- **营销触达**：强调"10分钟高效背奶"卖点
- **产品建议**：推荐下一代静音款产品

### 场景2: 群体画像生成（职场背奶妈妈）

**业务问题**  
Momcozy想针对"职场背奶妈妈"群体设计营销活动，需要深入理解这个人群的共性特征和差异化需求。

**数据输入**
```
筛选条件：
- 评论中提到"上班/公司/背奶/职场"
- 或购买场景标签包含"工作场景"
- 共5000+用户
```

**RAG检索与聚合**
```
检索该群体代表性评论（按主题聚类）：
噪音相关："在公司用声音大很尴尬"（高频）
便携相关："放包里很方便"（高频）
效率相关："10分钟搞定不耽误工作"（高频）
配件相关："零件太多容易丢"（中频）
```

**生成群体画像**
```json
{
  "segment_name": "职场背奶妈妈",
  "segment_size": "35% of user base",
  "demographics": {
    "age_range": "28-35岁",
    "occupation": "全职职场妈妈",
    "baby_age": "3-12个月"
  },
  "core_needs": [
    {
      "need": "高效吸奶",
      "importance": 9.2,
      "quote": "午休时间有限，必须快"
    },
    {
      "need": "静音体验",
      "importance": 8.8,
      "quote": "会议室隔壁就是老板办公室"
    },
    {
      "need": "便携性",
      "importance": 8.5,
      "quote": "每天背来背去不能太重"
    }
  ],
  "pain_points": [
    {
      "pain": "公司哺乳室条件差",
      "frequency": "62%",
      "solution": "强调随时随地可用"
    },
    {
      "pain": "吸奶时间与会议冲突",
      "frequency": "45%",
      "solution": "强调速度快"
    }
  ],
  "marketing_insights": {
    "key_message": "10分钟高效背奶，静音不尴尬",
    "channel": "小红书/微博职场妈妈社群",
    "timing": "产假结束前2周",
    "bundle_suggestion": "主机+便携包+备用配件"
  }
}
```

**业务价值**
- 营销转化率从2.3%提升至4.1%（+78%）
- 客单价从￥399提升至￥528（+32%，配件捆绑）
- 用户满意度从4.2提升至4.6星

## ③ 代码模板

代码位置: `paper2skills-code/nlp_voc/personabot_rag_profiling/model.py`

核心组件：
1. **ReviewRetriever**: RAG检索器（向量相似度）
2. **PersonaGenerator**: LLM画像生成器
3. **PersonaValidator**: 画像验证器（完整性/一致性）
4. **PersonaSchema**: 画像结构定义
5. **PERSONABOTProfiler**: 主流程整合

运行测试:
```bash
cd paper2skills-code/nlp_voc/personabot_rag_profiling
python3 model.py
```

## ④ 技能关联

### 前置技能
- **Skill-TopicImpact-观点单元画像抽取**: 提供细粒度观点输入
- **Skill-Spiral-of-Silence-沉默少数派挖掘**: 发现代表性评论
- **Skill-REVISION-无点击意图挖掘**: 补充搜索意图维度

### 延伸技能
- **Skill-SoMeR-多视角用户表示**: 嵌入化表示
- **Skill-Recommendation-Personalization**: 推荐应用
