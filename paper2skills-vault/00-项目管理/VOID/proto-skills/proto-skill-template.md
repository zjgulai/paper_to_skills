# 原型 Skill 写作模板

> **使用场景**：当问题银行里的 Level 1 问题成熟到可以启动初步研究时使用。
> **区别于标准 Skill**：不需要论文引用、不需要完整可运行代码、不需要精确 ROI 量化。
> **唯一要求**：有一个可以用真实数据在真实业务场景里产生真实摩擦的实验设计。
