## 三、MAS 多智能体系统 MVP

### 5 个端到端业务工作流(全部端到端可跑)
- **WF-A 智能补货**(SupplyChainAgent + restock_graph)
- **WF-B 广告优化**(MarketingAgent + ad_campaign_graph)
- **WF-C 客服分诊**(CustomerServiceAgent + customer_ops_graph)
- **WF-D 选品扫描**(SelectionAgent + selection_graph)
- **WF-E Review 监控**(复用 CustomerServiceAgent,workflow_type 路由)

### 系统结构
```
mas/
├── agents/           # 5 业务 Agent + Orchestrator + QA
├── graphs/           # StateGraph + 5 工作流
├── skills/           # SkillRegistry + 4 业务工具层
├── state/            # WorkflowContext TypedDict
├── hitl/             # FastAPI 风格审批接口
├── checkpointing/    # SQLite Saver(生产可换 Postgres)
├── observability/    # Tracer(生产可换 LangSmith)
└── tests/            # 37/37 集成测试全绿
```

### 测试覆盖
- 阶段 0(骨架)12/12 / WF-A 5/5 / WF-B 5/5 / WF-C/E 5/5 / WF-D 4/4 / 阶段 6(集成)6/6 = **37/37**
