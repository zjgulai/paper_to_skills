### 系统结构
```
mas/
├── agents/           # 5 业务 Agent + Orchestrator + QA
├── graphs/           # StateGraph + 5 工作流
├── skills/           # SkillRegistry + 4 业务工具层
├── state/            # WorkflowContext TypedDict
├── hitl/             # FastAPI 风格审批接口
├── checkpointing/    # SQLite Saver(生产可换 Postgres)
├── observability/    # Tracer(生产可换 LangSmith)
└── tests/            # 37/37 集成测试全绿
```
