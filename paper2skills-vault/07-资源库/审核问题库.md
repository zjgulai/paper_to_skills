# 审核问题库

记录 skill 审核中发现的常见问题模式，供后续审核参考。

## 常见问题分类

### 算法原理问题

| 问题类型 | 描述 | 解决建议 |
|---------|------|---------|
| 复制摘要 | 直接复制论文摘要而非重述 | 用自己的话重述核心思想 |
| 缺少公式 | 没有关键公式 | 添加核心公式和直觉解释 |
| 假设不明 | 未说明关键假设和使用条件 | 明确列出3-5个关键假设 |

### 应用案例问题

| 问题类型 | 描述 | 解决建议 |
|---------|------|---------|
| 场景不相关 | 案例与母婴出海无关 | 重新设计母婴出海相关案例 |
| 问题模糊 | 业务问题描述不具体 | 明确业务问题，包含具体指标 |
| 价值无依据 | 业务价值没有量化依据 | 添加预期收益的量化估算 |

### 代码模板问题

| 问题类型 | 描述 | 解决建议 |
|---------|------|---------|
| 占位符过多 | 代码中有 TODO 占位符 | 完成所有占位符的实现 |
| 无测试用例 | 没有测试代码 | 添加至少3个测试用例 |
| 输入输出不明 | 函数无输入输出定义 | 添加 docstring 和类型注解 |

### 技能关联问题

| 问题类型 | 描述 | 解决建议 |
|---------|------|---------|
| 无关联 | 没有关联其他 skill | 至少关联2个相关 skill |
| 逻辑不清 | 关联没有逻辑依据 | 说明关联的业务逻辑 |

### 商业价值问题

| 问题类型 | 描述 | 解决建议 |
|---------|------|---------|
| ROI 无依据 | ROI 预估没有计算依据 | 添加量化的计算逻辑 |
| 评分无说明 | 评分但没有具体说明 | 添加评分理由和依据 |

## 领域专属审核检查点（2026-05 扩展）

### 知识图谱 / 图神经网络（08-知识图谱）

| 问题类型 | 描述 | 解决建议 |
|---------|------|---------|
| 假设检验缺失 | 异构图/双曲嵌入论文未说明在母婴出海场景下的图结构假设 | 明确列出 node/edge type，给出在 SKU-用户-评论 三元图上的具体构建方式 |
| 评估指标错配 | 用 Hits@K 不带业务解读 | 同时给出 Hits@10 与"召回价格带正确的相关 SKU 比例"等业务指标 |
| 冷启动场景缺失 | KGQA / GraphRAG 类未说明新 SKU 上线零交互场景 | 必须给出冷启动 fallback 策略 |

### 多智能体系统（10-MAS）

| 问题类型 | 描述 | 解决建议 |
|---------|------|---------|
| Agent 角色模糊 | "多 Agent 协作"未明确各 Agent 的输入/输出契约 | 用表格列出每个 Agent 的 input schema / output schema / 失败语义 |
| 编排成本失控 | 未估算 token 消耗和 wall-clock 时间 | 用 N 次任务 × token/任务给出运营成本量级 |
| 自我反思无界 | Reflexion/Self-improvement 类未给停止条件 | 必须设置 max_iterations 与收敛判定 |

### 智能体工程（16-智能体工程）

| 问题类型 | 描述 | 解决建议 |
|---------|------|---------|
| Skill 边界不清 | "Auto Skill Synthesis" 未定义"什么是合格的 Skill" | 给出 Skill 输入/输出/前置条件/失败模式四元组规范 |
| MCP/A2A 协议落地缺失 | 仅讨论协议未给出 server/client 最小可运行示例 | 至少提供一个 Skill server 的 hello-world 实现 |
| 工具描述质量未评分 | "Tool Description Audit" 未给可执行评分函数 | 必须给出基于 LLM 或规则的可执行打分代码 |
| Context 管理无指标 | "Context Compression / Pruning" 未给压缩率与精度权衡 | 用 token_saved % vs task_success % 二维曲线展示 |

### 广告分析 / 营销投放（13-广告分析 / 15-营销投放分析）

| 问题类型 | 描述 | 解决建议 |
|---------|------|---------|
| 归因模型偏差未讨论 | Shapley/Markov 归因未说明位置偏差和选择偏差 | 必须列出已知偏差源与缓解方法（含 IV/RDD 等无偏估计可选项） |
| MMM 数据假设不明 | Marketing Mix Modeling 未声明饱和度函数与 carry-over | 显式给出 Hill saturation / Adstock 假设的业务合理性 |
| 因果 ML 反事实不可验证 | DML/CausalForest 缺少 holdout 实验对照 | 必须给出 A/B 实验或 holdout geo 验证设计 |
| ROAS 预算优化无约束 | 预算分配未考虑频次上限/创意疲劳 | 加入 frequency cap 与 fatigue penalty 约束 |

### 数据 Agent / LLM 分析（09-DataAgent-LLM）

| 问题类型 | 描述 | 解决建议 |
|---------|------|---------|
| SQL 生成无校验 | SQL Agent 未说明执行前的语法/语义校验 | 必须给出 EXPLAIN / dry-run 校验步骤 |
| 幻觉容忍度未声明 | LLM 数据分析未说明数值可信度阈值 | 给出"低置信结果必须人工 review"的边界条件 |
