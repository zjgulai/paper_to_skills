# Demand Forecasting 实施指南

## 场景
母婴用品周销量预测

## 技术架构

```
┌─────────────────┐     ┌──────────────────┐     ┌─────────────────┐
│   数据源        │────▶│  预测引擎        │────▶│  库存系统       │
│ (销售/库存/促销)  │     │ (Prophet+ML组合)   │     │ (安全库存计算)  │
└─────────────────┘     └──────────────────┘     └─────────────────┘
         │                        │
         ▼                        ▼
┌─────────────────┐     ┌──────────────────┐
│   特征工程      │     │   预测输出       │
│ (时序特征/日历) │     │ (点预测+区间)    │
└─────────────────┘     └──────────────────┘
```

## 快速开始

### 1. 数据准备

```python
import pandas as pd
from paper2skills_code.time_series.demand_forecasting import DemandForecaster

# 准备数据
df = pd.read_csv('sales_data.csv')
df['date'] = pd.to_datetime(df['date'])
df = df.sort_values('date')

# 划分训练/测试
train = df[df['date'] < '2024-01-01']
test = df[df['date'] >= '2024-01-01']

# 训练模型
forecaster = DemandForecaster(model_type='simple')
forecaster.fit(train['date'], train['sales'])

# 预测未来 4 周
predictions, lower, upper = forecaster.predict(n_periods=4)
```

## 模型选择

| 模型 | 适用场景 | 复杂度 | 精度 |
|------|----------|--------|------|
| Simple | 快速 POC | 低 | 中 |
| Prophet | 季节性明显 | 中 | 高 |
| Exponential Smoothing | 趋势稳定 | 低 | 中 |
| LSTM | 复杂模式 | 高 | 很高 |

## 注意事项

1. **数据质量**：历史数据准确性直接影响预测
2. **季节性**：母婴行业有大促季，需特殊处理
3. **新品**：新品需要特殊处理（参考同类产品）
4. **外部因素**：竞品活动、政策变化需手动调整
